# Apache CloudStack (apache-cloudstack)
Apache CloudStack is an open-source cloud computing software for creating, managing, and deploying infrastructure cloud services. It provides a comprehensive IaaS platform with support for multiple hypervisors and a rich API for cloud resource management.

### Apache CloudStack API
The CloudStack API provides comprehensive REST endpoints for managing virtual machines, networks, storage, accounts, domains, zones, and all other cloud infrastructure resources, with support for both synchronous and asynchronous operations.
